Episio

Project Overview
 -Episio is designed to help users organize and monitor various types of media such as movies, songs, anime, manga, and books. It allows users to add, uodate, and view entries, as well as track their progress (e.g., episodes, chapters, or volumes) and personal ratings. The system aims to provide a centralized platform for managing entertainment consumption efficiently.

Developers

Problem Statement
 -Many users consume different types of media daily but do not have a single system to track all of them efficiently. Most existing platforms are limited to one types of media, which makes tracking across multiple categories inconvenient and unorganized.

Objectives
1. To develop a system that allows users to track various types of media in one platform
2. To enable users to update their progress (episodes, chapters, volumes)
3. To allow users to rate and monitor their media consumption

Methodology
 -The system was developed using Python and implemented in Visual Studio Code. A menu-driven interface allows users to interact with the system easily.

The program uses fundamental programming concepts such as:
1. Functions
2. Conditional statements
3. Loops 
4. File handling

Use of Database
 -The system uses a NoSQL approach by storing data in a JSON file. JSON acts as a lightweight database where all entries are saved in a structured format.

Stored data includes:
1. Media Types
2. Title
3. Status
4. Rating
5. Episodes, chapters, or volumes
6. Date added

Data Collection and Analysis Plan
 -Data is collected through user input when adding or updating media entries. The system processes this date to:
1. Display organized lists of media
2. Track user progress
3. Compute average ratings
4. Monitor total entries

Feasibility and Timeline
 -The project is highly feasible as it uses basic programming concepts and lightweight data storage

Innovation and Creativity
 -The system introduces flexibility by allowing users to define their own media types instead of limiting them to predefined categories. It also supports datailed progress tracking (episodes, chapters, volumes), making it adaptable for various forms of entertainment.
